Color Theme Customization

To match your brand colors, add these CSS custom properties:

<style>
:root {
    --primary-color: #your-brand-color;
    --primary-hover: #your-brand-hover-color;
    --bg-dark: #your-dark-bg-color;
    --bg-secondary: #your-secondary-bg-color;

Backend Configuration

1. Update API Endpoint

If your backend is running on a different domain, update the apiEndpoint in the JavaScript file:

// In chat-widget.js, update this line:
this.apiEndpoint = 'https://your-backend-domain.com/chat';

2. Update Cal.com Link

In the chat-widget.js file, find the createDemoButton() method and update the Cal.com URL:

button.href = 'https://cal.com/your-actual-calendar-link';

Quick Test

Testing Your Integration

  1. Open your webpage in a browser
  2. Look for the chat bubble in the bottom-right corner
  3. Click the bubble to open the chat panel
  4. Type a message and press Enter
  5. The AI should respond within a few seconds

Troubleshooting

Common Issues

  • Chat bubble not showing: Check console for JavaScript errors
  • No AI responses: Verify backend is running and API key is set
  • Styling issues: Ensure CSS file is loaded correctly
  • CORS errors: Update CORS settings in backend
